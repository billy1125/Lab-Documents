## 說明

本 repository 收錄實驗室運作所需的基本規範與說明文件，目的是讓所有成員在同一套明確的前提下進行研究與日常運作。

請理解以下幾點：

- 規範的存在，是為了降低溝通成本與避免不必要的誤解  
- 規範不會涵蓋所有情境，但提供基本行為準則  
- 若規範與實際情況有落差，應主動討論，而非自行解讀或忽略  

## 閱讀對象

本文件適用於：

- 尚未加入但正在考慮的同學  
- 新加入實驗室的同學  
- 已在實驗室中的成員  

## 重要提醒

- 規範會隨時間或實驗室狀況調整  
- 不會逐一通知每一次變更  
- 請自行定期回來確認最新版本  

## 基本共識

在這個實驗室中，我們希望建立的不是高度監控的制度，而是：

> **在清楚規範下，能夠自我管理與負責的環境**

這代表：

- 你需要主動理解規範，而不是被動等待提醒  
- 有問題應提出，而不是自行假設  
- 規範是最低標準，而不是目標  

## 最後

如果你對任何規範內容有疑問或覺得不合理：

- 請直接提出討論  
- 多數問題可以透過溝通解決  

但前提是：

> **你已經閱讀過相關文件，並嘗試理解內容。**
