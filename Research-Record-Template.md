# 研究紀錄模板

> 使用方式：每次重要研究活動請複製本模板並填寫  
> 適用範圍：AI、管理、社會科學研究  
> 說明：可依研究類型略作調整，但請保留主要結構

## 基本資訊

- 日期：
- 負責人：
- 研究主題：
- 紀錄編號（可選）：
- 研究類型（AI / 管理 / 社會科學）：

## 一、研究問題或目標

請清楚說明：

- 這次要做什麼？
- 這次想回答什麼問題？
- 為什麼需要做這件事？

## 二、背景與依據

請說明本次工作的背景：

- 與前一次研究的關係
- 參考的文獻、理論或先前結果
- 為何選擇這個方向

## 三、方法或研究設計

請說明本次使用的方法：

- 方法名稱：
- 方法說明：
- 選擇原因：
- 若有替代方案，為何未採用：

## 四、資料 / 材料 / 樣本

請依研究類型填寫相關內容。

### 通用欄位

- 名稱：
- 來源：
- 版本 / 時間：
- 使用範圍：
- 備註：

### 若為 AI / 工程研究

- Dataset：
- 前處理方式：
- 環境設定：
- 模型 / 系統版本：

### 若為管理研究

- 資料來源（報告 / 個案 / 問卷 / 次級資料）：
- 分析單位：
- 資料期間：

### 若為社會科學研究

- 樣本來源：
- 樣本數：
- 篩選條件：
- 問卷 / 訪談 / 實驗設計版本：
- 倫理或匿名處理說明：

## 五、執行過程

請記錄實際做了哪些事：

- 執行步驟
- 中途調整
- 遇到的問題
- 當下的判斷與處理方式

必要時可條列說明。

## 六、研究設定補充

### 若為 AI / 工程研究，請補充：

- Git repo：
- Branch：
- Commit ID：
- 主要參數：
- 硬體環境：

### 若為管理研究，請補充：

- 理論框架：
- 分析方法：
- 推論邏輯：

### 若為社會科學研究，請補充：

- 研究設計類型：
- 資料收集方式：
- 資料清理方式：
- 統計 / 分析方法：

## 七、結果

### 1. 主要結果

請描述本次得到的主要發現：

- 定量結果：
- 定性觀察：
- 是否符合預期：

### 2. 附加輸出（如有）

- 圖表：
- 表格：
- 模型輸出：
- 訪談摘要：
- 統計結果：

## 八、問題與觀察

請誠實記錄：

- 哪些地方不理想？
- 哪些地方有風險？
- 哪些地方需要重新確認？
- 本次結果可能受哪些因素影響？

## 九、結論

請簡要說明：

- 本次工作是否有效？
- 回答了哪些問題？
- 還有哪些問題尚未回答？

## 十、下一步

請具體寫出下一步：

- 下一個要做的研究或分析
- 要補的資料
- 要修正的方法
- 預計產出

## 十一、相關檔案位置

請記錄相關資源位置：

- 文件位置：
- 資料位置：
- 程式碼位置：
- 輸出位置：

## 十二、提交前檢查

請確認：

- [ ] 有寫清楚研究問題或目標
- [ ] 有寫清楚方法或研究設計
- [ ] 有記錄資料 / 材料 / 樣本
- [ ] 有記錄執行過程
- [ ] 有結果
- [ ] 有問題與觀察
- [ ] 有下一步
- [ ] 有相關檔案位置

## 補充說明

- 不需要寫得很長，但需要讓他人看得懂
- 若一次工作內容很多，可拆成多份紀錄
- 若研究類型有特殊需求，可增加欄位，但不要刪除核心欄位

> 這份模板的目的，是讓不同研究類型都能在同一套管理架構下累積與延續。
